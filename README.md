# droid-autoresearch-marketplace

Public Droid marketplace for the `droid-autoresearch` plugin.

## What this marketplace contains

- `droid-autoresearch` — a Droid plugin for autonomous benchmark-driven improvement loops with persistent playbooks, curated experiment memory, and resume-aware hooks.

## Install

```powershell
droid plugin marketplace add https://github.com/alertemikope/droid-autoresearch-marketplace
droid plugin install droid-autoresearch@droid-autoresearch-marketplace --scope project
```

For user scope:

```powershell
droid plugin install droid-autoresearch@droid-autoresearch-marketplace --scope user
```
